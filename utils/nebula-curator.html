⚙ setup — read this once

This tool uses the Anthropic API to analyze URLs and assign them to constellations. You need an API key to use it. Get one at console.anthropic.com → API Keys → Create Key. It starts with sk-ant-...

Your key is stored only in this browser's localStorage — it never leaves your machine except to call the Anthropic API directly. It is not sent to perceptualmechanics.com or anywhere else.

This page is password-protected so only you can access it. Still, don't share the URL with your key visible in DevTools to anyone.

To use: paste your API key below, click Save. Then paste YouTube (or any) URLs one at a time. Claude will name each star and assign it to a constellation. When you have enough stars, hit "rebuild constellations" to let Claude remap everything thematically. When you're happy, hit "copy as JS" and paste the output into src/scenes/nebula.js.
